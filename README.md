# visitkorea-project
대한민국 구석구석 사이트([Visit Korea](https://korean.visitkorea.or.kr)) 클론 코딩 프로젝트

### 팀원
김효준 손혜정

### 기술 스택
- HTML, CSS (Vanilla)
- JavaScript
- Git & GitHub

### Commit Message Conventions

| 타입 | 설명 |
| --- | --- |
| `feat` | 새로운 기능 추가 |
| `fix` | 버그 수정 |
| `docs` | 문서 수정 (README 등) |
| `style` | 코드 포맷팅 (기능/로직 변경 없음) |
| `refactor` | 리팩토링 |
| `test` | 테스트 코드 추가/수정 |
| `chore` | 빌드/패키지/환경 설정 등 잡일 |
